# XRD Scorer v1.0.0

A professional desktop application for X-ray Diffraction (XRD) data analysis with advanced processing capabilities.

## Quick Start

### Installation

1. Download the executable from [GitHub Releases](https://github.com/srinikhil25/releases)
2. Extract and run `XRD_Scorer.exe`
3. No additional installation required

### Basic Usage

1. **Load File**: Click "Load File" and select your XRD data file (XRDML, DAT, ASC, TXT, or RAW format)
2. **Process Data**: 
   - Apply background subtraction (recommended: Iterative Polynomial)
   - Apply K-alpha stripping (optional)
3. **Analyze Peaks**: Click "Detect Peaks" to identify diffraction peaks
4. **Match Reference**: Search and overlay reference patterns from ICDD/MP databases
5. **Save Project**: All analysis steps are automatically saved to the `projects/` folder

## Features

- ✅ Multi-format file support (XRDML, DAT, ASC, TXT, RAW)
- ✅ Five background subtraction algorithms
- ✅ K-alpha stripping (Rachinger correction)
- ✅ Advanced peak detection with prominence filtering
- ✅ Reference pattern matching (ICDD & Materials Project)
- ✅ Publication-ready visualizations
- ✅ Automatic project management for AI integration
